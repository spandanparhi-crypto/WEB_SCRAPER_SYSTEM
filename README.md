# 🕷️ Web Scraper System

A modern web-based **Web Scraper System** that allows users to extract publicly available information from websites by entering a URL. The application fetches webpage content, displays the extracted data in a structured format, and enables users to search, preview, copy, and download the extracted results through a clean and responsive interface.

# 📖 How to Use

### Step 1

Open the application.

### Step 2

Enter a valid website URL.

### Step 3
